Eritropoyesis y Regulación

  • Célula madre pluripotente (médula ósea) → BFU-E → CFU-E → pronormoblasto
  • Maduración: normoblasto basófilo → policromatófilo → ortocromático → reticulocito (aún ARN) → eritrocito maduro (bicóncavo, sin núcleo)
  • EPO: riñón (90%) en hipoxia → receptor EPO → ↑proliferación/diferenciación CFU-E → maduración acelerada
  • Cofactores esenciales: Hierro (Hb), B12 (división celular), Folato (síntesis ADN), Vitamina C (absorción Fe)
Ciclo del Hierro

Vida eritrocito ≈120 días → destrucción bazo/hígado → hemoglobina → hemo → bilirrubina (conjugada → bilis) + hierro reciclado → transferrina → médula ósea

Anemia Ferropénica: La Más Frecuente

Fisiopatología por Estadios

Depleción reservas (ferritina↓) → deficiencia eritropoyesis (TIBC↑, sat transferrina↓, ZPP↑) → anemia manifiesta (Hb↓, microcitosis, hipocromía, anulocitos)

Diagnóstico

Hb <13 g/dL (H) / <12 g/dL (M). VCM <80 fL. Ferritina <30 ng/mL (definitivo). Hierro sérico↓, TIBC↑, Sat transferrina <16%. Frotis: microcitos, hipocromía, anulocitos, poiquilocitosis

Tratamiento

VO: Sulfato ferroso 325 mg 3×/día en ayunas + Vitamina C. Duración: 3 meses tras normalizar Hb (reponer depósitos).
IV: hierro carboximaltosa si intolerancia VO, malabsorción, ERC, cirugía programada

4

Anemia Megaloblástica: B12 y Folato

Mecanismo Común

B12 y Folato → síntesis timidina → replicación ADN → eritroblastos grandes (megaloblastos) → hemólisis intramedular → anemia + neutrófilos hipersegmentados (>5 lóbulos)

Déficit B12 (Cobalamina)

Causas: anemia perniciosa (anti-FI/anti-célula parietal), gastrectomía, vegetarianismo estricto, metformina crónica. Absorción: B12+FI → íleon terminal.
Síntomas neurológicos: mielosis funicular (parestesias, ataxia, demencia).
Tx: B12 IM 1mg/día ×7 días → mensual de por vida

Déficit Folato

Causas: dieta pobre (alcohólicos), embarazo (↑requerimiento), antifolatos (metotrexato, trimetoprim, fenitoína), malabsorción.
Sin síntomas neurológicos (diferencia clave de B12).
Tx: Ácido fólico 1-5 mg VO/día. Suplementación preconcepcional 400 μg (previene defectos tubo neural)

Anemias Hemolíticas

Hemólisis Intravascular

Destrucción dentro del vaso → hemoglobinuria (orina roja/marrón), hemoglobinemia, ↓haptoglobina. Causas: incompatibilidad ABO, CID, prótesis valvular, esferocitosis severa

Hemólisis Extravascular

Macrófagos esplénicos → ↑bilirrubina indirecta + ictericia + esplenomegalia + urobilinógeno urinario↑. Causas: esferocitosis hereditaria, drepanocitosis, talasemia

Hemostasia Primaria: Tapón Plaquetario

Secuencia Hemostasia Primaria

Daño endotelial → exposición colágeno → vWF (plasma) une colágeno + Gp1b plaqueta → adhesión → activación → liberación ADP/TXA2 → ↑reclutamiento → Gp IIb/IIIa une fibrinógeno → agregación → tapón plaquetario

vWF

Proteína multimérica (endotelio/plaquetas). Esencial en alto shear stress (arterias). Déficit: Enf. vWF tipo 1/2/3 → sangrado mucosas, menorragia

Activación Plaquetaria

Agonistas: trombina, ADP (P2Y12), TXA2 (COX-1), colágeno. Inhibidores fisiológicos: PGI2 (prostaciclina) + NO endoteliales

Antiagregantes

AAS (COX-1 irreversible → ↓TXA2). Clopidogrel/ticagrelor (P2Y12). Abciximab (GpIIb/IIIa). Indicaciones: SCA, stent, ACV isquémico

7

Cascada de Coagulación

  • Vía extrínseca: Daño tisular → Factor Tisular (FIII) + FVIIa → FXa
  • Vía intrínseca: Contacto (FXIIa) → FXIa → FIXa + FVIIIa → FXa
  • Vía común: FXa + FVa (protrombinasa) → Trombina (FIIa) → Fibrinógeno → Fibrina → FXIIIa → fibrina estable (coágulo)

TP/INR (extrínseca)

Mide FVII, FX, FV. Normal 11-14 s, INR 0.8-1.2. ↑en: warfarina, déficit Vit K, hepatopatía. Meta: INR 2-3 (FA/TVP) o 2.5-3.5 (prótesis mecánica)

TTPA (intrínseca)

Mide FXII,XI,IX,VIII,X,V,II. Normal 25-40 s. ↑en: hemofilia A (FVIII) / B (FIX), heparina no fraccionada, anticoagulante lúpico

Dímero-D

Producto fibrinólisis. Normal <0.5 μg/mL. ↑en: CID, TEP, TVP, embarazo, sepsis, cirugía. Alta sensibilidad (descarta TEP si baja prob + dímero-D normal)

8

Anticoagulantes y CID

Anticoagulantes

HNF: monitorizar TTPA (meta 1.5-2.5×). Reversión: protamina.

HBPM: sin monitoreo habitual (anti-Xa si obeso/embarazo). Reversión: protamina parcial.

NACO: Rivaroxabán/apixabán (anti-Xa). Dabigatrán (anti-IIa). Reversión: andexanet alfa / idarucizumab.

Warfarina: INR 2-3. Reversión: Vit K IV ± PFC.

CID — Coagulación Intravascular Diseminada

Trombosis microvasculares + consumo factores + plaquetas → sangrado paradójico.

Causas: sepsis, trauma masivo, LPA (M3), abruptio placentae, hemólisis transfusional.

Tx: tratar causa + PFC (factores) + plaquetas + crioprecipitado (fibrinógeno). TP/TTPA↑, Ddímero↑, fibrinógeno↓, plaquetas↓

⚠️

Trombofilias hereditarias frecuentes: Factor V Leiden (resistencia proteína C activada), déficit proteína C/S, déficit ATIII, mutación protrombina G20210A → TEP/TVP recurrentes

Hematooncología: Leucemias y Linfomas

Leucemia Aguda

Bloqueo maduración → acumulación blastos. LMA: M3=LPA (ATRA+ATO). LLA: ↑en niños, CR 80-90%. Clínica: anemia + infecciones + sangrado (pancitopenia). Blastos >20% médula

Leucemia Crónica

LMC: BCR-ABL (cromosoma Filadelfia t(9;22)). Tx: imatinib (ITK) respuesta >80%.
LLC: linfocitos B clonales maduros. >65 años, indolente. Tx: ibrutinib, venetoclax. Infección frecuente

Linfoma Hodgkin

Célula Reed-Sternberg (CD15+CD30+). Jóvenes. Estadios Ann Arbor. ABVD (adriamicina/bleomicina/vinblastina/dacarbazina). Curación >85%

Linfoma No Hodgkin

Heterogéneo. DLBCL (agresivo, curable): R-CHOP. Folicular (indolente, recidivante). Síntomas B: fiebre/sudoración nocturna/pérdida peso >10%

Rol de Enfermería — Hematología

Anemias — Evaluación

Palidez (mucosas/conjuntivas/palmas), fatigabilidad, disnea esfuerzo, taquicardia compensadora. Ferropenia: queilitis angular, koiloniquia, pica, glositis. B12: alteraciones neurológicas, glositis Hunter

Hemostasia — Monitorización

Petequias/equimosis/hematomas espontáneos (↓plaquetas/disfunción). Sangrado mucosas, epistaxis, menorragia. Control INR en anticoagulados: registro, ajuste dosis, interacciones (AINEs, ATB, alimentos ricos Vit K)

Transfusión Sanguínea

Doble verificación ABO/Rh + identidad paciente. 1 unidad GR en 90-120 min. Reacción transfusional: fiebre/escalofrío → suspender, vía nueva, avisar. Vigilar TACO (sobrecarga) y TRALI

Neutropenia Febril

ANC <500 + T° ≥38.3°C o ≥38°C ×1h → ATB empírico inmediato (β-lactámico antipseudomonas, máx 1h). Precauciones: mascarilla, sin frutas/verduras crudas, restricción visitas. Score MASCC: estratificación riesgo

Resumen — Hematología y Hemostasia

  • Eritropoyesis: EPO (hipoxia renal) → CFU-E → reticulocito → eritrocito. Vida ≈120 días
  • Microcítica: Ferropenia (ferritina↓, TIBC↑). Macrocítica: B12/Folato → megaloblastos + neutrófilos hipersegmentados. B12 = síntomas neurológicos (mielosis funicular)
  • Hemólisis intravascular: hemoglobinuria. Extravascular: bilirrubina↑, esplenomegalia
  • Hemostasia 1°: Colágeno → vWF → Gp1b → activación → Gp IIb/IIIa + fibrinógeno → tapón. AAS/clopidogrel la inhiben
  • Cascada: extrínseca (TP/INR) + intrínseca (TTPA) → vía común → fibrina estable
  • CID: trombosis microvascular + consumo factores → sangrado. Tratar causa + PFC + crioprecipitado
  • Neutropenia febril (ANC <500 + fiebre): ATB antipseudomonas en <1 hora (emergencia oncológica)
