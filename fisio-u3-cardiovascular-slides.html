Insuficiencia Cardíaca: Fisiopatología

Cascada de la IC

Injuria miocardio → ↓GC → activación SNS + SRAA → vasoconstricción + retención Na⁺/H₂O → ↑precarga/postcarga → remodelado ventricular (dilatación/hipertrofia excéntrica) → progresión IC.

IC CON FEVIr (FE <40%)

  • Disfunción sistólica
  • Dilatación VE
  • GC↓
  • Congestión pulmonar y sistémica
  • Causas: IAM, miocardiopatía, miocarditis

IC CON FEVIp (FE ≥50%)

  • Disfunción diastólica
  • Músculo rígido
  • Llenado ↓
  • Disnea de esfuerzo
  • Causas: HTA, hipertrofia, isquemia

  • VMNI (CPAP)

Shock: 4 Tipos y Fisiopatología

HIPOVOLÉMICO

↓Volumen → ↓GC

  • Hemorragia/deshidratación
  • Taquicardia
  • PVC baja
  • Vasoconstricción

CARDIOGÉNICO

↓Contractilidad → ↓GC

  • IAM masivo
  • Arritmia grave
  • PVC alta
  • Estertores

DISTRIBUTIVO

Vasodilatación masiva

  • NO↑ (séptico)
  • ↓RVP
  • Distribución inadecuada
  • Piel caliente

OBSTRUCTIVO

Obstáculo al flujo

  • Neumotórax tensión
  • TEP masivo
  • Taponamiento cardíaco
⚠
Shock séptico: sepsis + vasopresores para PAM≥65 + lactato>2 mmol/L sin hipovolemia (Sepsis-3, 2016).
4

Manejo Shock Séptico — Hour-1 Bundle

  • Hemocultivos ×2 (antes ATB) + Lactato sérico
  • Antibióticos IV de amplio espectro (sin demora)
  • Cristaloides 30 mL/kg si hipotensión o lactato ≥4 mmol/L
  • Vasopresores si PAM <65 mmHg pese a reposición → Norepinefrina 1° elección
  • Reevaluación hemodinámica (dinámicos: PLR, VPP)

Scoring Sepsis

SOFA score ≥2 = disfunción orgánica. qSOFA: FR≥22 + Glasgow<15 + PAS≤100 → ≥2 = screening sepsis fuera UCI.

5

Ateromatosis: Patogenia de la Placa

  • Disfunción endotelial: HTA, DM, tabaco, dislipidemia → ↓NO → adhesión leucocitos
  • Entrada LDL: LDL oxidado (oxLDL) → captación por macrófagos → células espumosas
  • Estría lipídica: Acumulación lípidos + foam cells
  • Placa fibrosa: Migración CML → cápsula fibrosa sobre núcleo lipídico
  • Placa vulnerable: Cápsula delgada + núcleo grande → ruptura → trombo → SCA

FACTORES DE RIESGO

Modificables:

  • HTA, DM, tabaco
  • Dislipidemia, obesidad
  • Sedentarismo

ESTABILIZACIÓN

  • Estatinas: ↓LDL + efecto antiinflamatorio
  • PCSK9i (evolocumab)
  • AAS
6

HTA: SRAA y Daño Órgano Blanco

Cascada SRAA

Renina (riñón isquémico/SNS) → Angiotensinógeno → Ang I → ECA → Ang II → Vasoconstricción + Aldosterona → Na⁺ retenido → ↑Volemia → ↑PA.

DAÑO ÓRGANO BLANCO

  • Cerebro: AIT/AVE hemorrágico
  • Corazón: HVI → IC diastólica → FA
  • Riñón: nefroangioesclerosis → ERC
  • Retina: retinopatía K-W
  • Aorta: disección

CRISIS HIPERTENSIVA

  • Urgencia: PA>180/120 sin DOB → VO
  • Emergencia: con DOB → IV titratable
  • Opciones: captopril, labetalol, nitroprusiato

8

IAM: Rotura de Placa y Oclusión

  • Placa vulnerable: cápsula delgada + núcleo lipídico grande + inflamación
  • Ruptura/erosión: exposición colágeno → adhesión plaquetaria
  • Agregación plaquetaria: TXA2, ADP → trombo blanco → oclusión
  • Oclusión coronaria: isquemia → lesión → necrosis (si >20-30 min)
  • Frente de necrosis: subendocardio → transmural (IAMCEST)

Zonas de lesión

Necrosis (irreversible) → Zona de lesión → Isquemia (recuperable con reperfusión). Tiempo = miocardio.

9

IAM: Diagnóstico y Manejo IAMCEST

DIAGNÓSTICO

Clínica:

  • Dolor precordial >20min
  • Irradiación brazo/mandíbula
  • Diaforesis, disnea

ECG:

  • BCRI nuevo
  • SDST≥1mm ×2 derivaciones

MANEJO REPERFUSIÓN

  • ICP primaria <90min (ideal)
  • Fibrinólisis <120min si no ICP
  • MONA: Morfina, O₂ (SpO₂<90%), Nitratos, AAS 300mg
  • P2Y12: ticagrelor/clopidogrel
  • Troponina: ↑ 3h, pico 12-24h
⚠
Regla general: tiempo = miocardio. Cada minuto de demora = 1.000 células muertas.

Rol de Enfermería — Sistema Cardiovascular

IC Y MONITORIZACIÓN

  • Peso diario (↑>1kg/día → alerta)
  • Balance hídrico estricto
  • Restricción sal 2g/día
  • FC/ritmo (FA frecuente)
  • SpO₂, posición Fowler 45°

SHOCK

  • SOFA/qSOFA
  • Acceso venoso 2 vías gruesas
  • Monitorización invasiva (PVC/PA)
  • Balance horario
  • TA cada 15min, diuresis >0.5ml/kg/h

IAM

  • 12 derivaciones inmediato
  • TIMI/GRACE score
  • Prep. ICP (rasurado, consentimiento)
  • Posición semi-Fowler
  • Ansiedad del paciente/familia

HTA

  • Toma PA correcta
  • Adherencia medicación
  • Dieta DASH
  • Restricción sodio
  • Actividad física

Resumen — Fisiopatología Cardiovascular

  • IC-FEVIr (<40%): sistólica. IC-FEVIp (≥50%): diastólica. NYHA I-IV
  • Pilares IC-FEVIr: IECA/ARA-II + BB + espironolactona + SGLT2i
  • Shock: Hipovolémico (↓vol) / Cardiogénico (↓contractilidad) / Distributivo (↓RVP) / Obstructivo
  • Hour-1 bundle: cultivos→ATB→fluidos→vasopresores→reevaluación. Norepinefrina 1ª elección
  • Ateromatosis: LDL oxidado → foam cells → placa → ruptura → SCA
  • SRAA: Renina→Ang II→vasoconstricción + aldosterona→↑volemia→HTA
  • HTA DOB: Cerebro/corazón/riñón/retina/aorta. Emergencia = con DOB
  • IAM: tiempo-miocardio. ICP<90min. Troponinas pico 12-24h. AAS + P2Y12
